# 原地修改
27.移除元素
- 把数组想象成一个栈,满足条件入栈,不满足就跳过
- 时间复杂度O(n),n为数组长度,空间复杂度O(1)

2273.移除字母异位词后的结果数组
- 这题就是你先拿出来数组中的第一个数组元素base,对他进行排序,排序完就遍历剩下元素,下标从1开始,每次循环对每个剩下元素都进行排序,并把排序后的结果与第一个元素比较(**equals**),相同则跳过,不相同就把右边元素存到base,并且把当前字符串元素存到words数组,最后返回Arrays.asList(Arrays.copyOf(words, k)),意思是把words数组的前k个元素作为List返回  
- 时间复杂度O(nL) 或者 O(nLlogL)，其中 n 是 words 的长度，L≤10 是 words[i] 的最大长度
- 空间复杂度：O(L)

3684.至多K个不同元素的最大和
- 先把数组排序, sort完之后就去除重复元素(力扣26),nums的长度大于K则返回前K个数,小于就返回nums数组,创建ans数组,然后nums从后往前赋值给ans,因为题目要求最大和
- 时间复杂度O(nlogn) 或 O(nlogk)，其中 n 是 nums 的长度。瓶颈在排序上
- 空间复杂度O(n) 或 O(1)。原地去重则空间复杂度为 O(1)，忽略排序的栈开销，返回值不计入
**去重逻辑**👇
``` java
private int remove(int[] nums) {
        int k = 1;
        for(int i = 1; i < nums.length; i++) {
            if(nums[i] != nums[i - 1]) {
                nums[k++] = nums[i];
            }
        }
        return k;
    }
```

283.移动零
- 从左到右遍历数组,同时维护一个左边界i0,遍历时交换零与非零的位置,因为题目要求原数组顺序不能改变,如此一来遍历完时(i0, n - 1)就全是零,同时保证原数组的顺序
- 时间复杂度O(n),n为数组长度,空间复杂度O(1)

905.按奇偶排序数组
- 双指针,维护i与j,从左找到第一个不为偶数的奇数,从右找到第一个不为奇数的偶数,交换这两个的位置
- 时间复杂度O(n),n为数组长度,空间复杂度O(1),没有用到额外空间

922.按奇偶排序数组II
- 所有偶数下标记为集合A,奇数下标记为集合B,集合A即nums[0], nums[2]... 集合B为nums[1], nums[3],分别遍历集合A,B,即每次+=2,在集合A中找到第一个奇数元素,集合B中找到第一个偶数元素,交换位置,实现原地修改,以此类推.
- 时间复杂度O(n),两个集合加起来遍历了一个数组的长度,空间复杂度O(1),没有额外空间

3467.将数组按照奇偶转化
- 暴力法就是直接for循环赋值然后sort排序,但是这题有巧妙写法,建立一个0,1cnt数组,用来记录0(偶数)和1(奇数)的个数,然后用到了Arrays.fill()方法,填充0和1,满足题目非递增需求.
- 时间复杂度O(n),n为数组长度,空间复杂度O(1),没有用到额外空间
关于Arrays.fill()
1. 全部填充: Arrays.fill(Object[] arr, Object val)
2. 部分填充: Arrays.fill(Object[] arr, int start, int end, Object val),左闭右开

2460.对数组执行操作
- 这题直接模拟即可,非零元素移动到数组头部即可,两个指针i和j,i用来遍历,j用来标记头部位置.
- 时间复杂度为O(n),n表示数组长度,只用遍历一遍,空间复杂度O(1),直接在原数组修改,无额外空间
