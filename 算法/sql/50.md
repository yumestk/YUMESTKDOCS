#### Write your MySQL query statement below
``` sql
select customer_id, count(customer_id) as count_no_trans from visits left join transactions using(visit_id) where transaction_id is null group by customer_id;
```
语法糖
using(visit_id) 等价于 on visits.visit_id = transactions.visit_id

GROUP BY 是为了和聚合函数配合用的
像 count()、sum()、max() 这些聚合函数，必须配合 GROUP BY 才能按组统计，不然数据库不知道你是想统计所有行，还是每个顾客单独统计。
SELECT 里非聚合的列，必须出现在 GROUP BY 里
比如你 SELECT 了 customer_id，那它必须在 GROUP BY 里，不然 SQL 会报错。
