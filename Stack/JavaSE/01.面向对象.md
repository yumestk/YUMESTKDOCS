什么是 Java 的封装特性？
- 封装就是把数据和操作数据封装在一起,对外只暴露必要的接口,内部实现细节不让外部直接碰.Java通过访问修饰符实现封装,字段用private藏起来,通过public的getter/setter方法控制访问
- 封装的核心价值在于控制,你控制了数据怎么被读取,怎么被修改,就控制住了数据的正确性

什么是 Java 中的继承机制？
- 允许子类复用父类的字段和方法,用extends关键字建立继承关系.Java只支持单继承,一个类最多一个直接父类,但可以通过继承链形成多层结构

什么是 Java 的多态特性？
- 父类引用指向子类对象,调用同一个方法时,会执行子类自己的实现
- 一条指令,多种表现,编译看左边,运行看右边,底层是通过虚方法表vtable动态绑定找到真正要执行的方法
- 多态的三个条件:要有继承或实现关系,extends or implement; 子类要重写override父类方法;父类引用指向子类对象,也叫向上转型
