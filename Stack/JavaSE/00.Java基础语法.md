**Java 中的基本数据类型有哪些？**
- 8种基本数据类,按用途分为四类: 整型,浮点型,字符型,布尔型
1. 整型:byte 1字节存小数字(-128, 127), short 2字节基本没人用(-32768, 32767), int 4字节(+-21亿), long 8字节用来存大数字(+-922亿亿),赋值要加L后缀
2. 浮点型: float占4字节要加F后缀, double占8字节精度更高,是默认的小数类型
3. char:占两字节存一个Unicode字符,底层是一个无符号整数,范围(0, 65535)
4. boolean表示真假,JVM没定死大小,通常占1个字节


**Java 中的参数传递是按值还是按引用？**
- 参数传递只有按值传递, 不论是基本类型还是引用类型
1. 基本类型比如int这些,传递的是值的副本,也就是数值本身,对方法参数的任何修改都不会影响原始变量
2. 引用数据类型比如对象,数组,传递的是引用的副本,也就是对象引用的内存地址,方法内可以通过引用修改对象的属性,但不能改变引用本身,使其指向另一个对象
