MySQL 中 count (*)、count (1) 和 count (字段名) 有什么区别
- 三者都是统计行数的聚合函数,核心区别在于对 NULL 值的处理方式不同
- count(*) 和 count(1) 会统计所有行,包含NULL值的行; count(字段名)只统计字段不为NULL的行数
- count(*) 和 count(1) 的效率完全一样count(字段名)如果字段没索引就是全表扫描,还要额外判断是否为NULL,所以比前者慢一点,如果字段是主键或者有索引,差距就不大了
